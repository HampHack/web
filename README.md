# HampHack Web
## Website for HampHack

🌐 - [Visit](https://hamphack.hampshire.edu)

## Contributing

Clone the repository:

```bash
    $ git clone https://github.com/HampHack/web.git
```

Install packages:

```bash
    $ yarn
```

_OR_

```bash
    $ npm install
```

Start:

```bash
    $ yarn dev
```

_OR_

```bash
    $ npm run dev
```
